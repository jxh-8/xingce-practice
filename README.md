# 行测刷题系统

公务员考试行测备考工具，支持预习、刷题、模拟考试、错题管理。

## 功能

### 知识预习
- 常识判断、言语理解、判断推理、数量关系、资料分析
- 每个模块的知识点、解题方法、高频考点

### 专项刷题
- 每模块15-16道精选题目
- 随机抽取10题练习
- 答题后即时显示解析

### 模拟考试
- **行测全真模拟**：120分钟倒计时，100题（每模块20题）
- **模块限时测试**：自选模块和时间
- 倒计时提醒，到时间自动交卷
- 答题卡快速跳转

### 成绩追踪
- 成绩趋势图，可视化分数变化
- 历史考试记录

### 错题本
- 自动记录错题
- 搜索、筛选、排序
- 标记已掌握/未掌握
- 错题检测（随机10题）
- 导出为TXT文件

### 手写草稿纸
- 右下角 ✏️ 按钮打开
- 支持手写笔和鼠标
- 画笔颜色、粗细可调
- 撤销、清空功能
- 可拖动位置

## 使用方法

### 本地运行
直接双击 `index.html` 用浏览器打开即可。

### 局域网访问（平板/手机）
```bash
cd 行测刷题系统
python -m http.server 8080
```
然后在平板浏览器访问 `http://你的IP:8080`

## 技术栈
- 纯 HTML/CSS/JavaScript
- 无外部依赖，单文件应用
- 数据存储在浏览器 localStorage

## 题目说明
题库为根据国考行测题型生成的示例题，用于功能演示和基础练习。建议配合粉笔、华图等平台的真题使用。
